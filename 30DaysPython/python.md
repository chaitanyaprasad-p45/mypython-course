### 1. Introduction
Overview of Python, its history, and why Python is popular.

### 2. Variables, Built-in Functions
- Variables store data in memory.
- Python has dynamically typed variables.
- Example:
  ```python
  x = 10  # Assigning an integer value
  name = "Alice"  # Assigning a string value
  print(type(x), type(name))  # Checking data types
  ```

### 3. Operators
- Arithmetic (`+`, `-`, `*`, `/`), logical (`and`, `or`), and comparison (`==`, `!=`, `>`).
- Example:
  ```python
  a, b = 10, 5
  print(a + b)  # Addition
  print(a > b)  # Comparison
  ```

### 4. Strings
- Immutable sequences of characters.
- Example:
  ```python
  text = "Python"
  print(text.upper())  # Convert to uppercase
  ```

### 5. Lists
- Ordered, mutable collections.
- Example:
  ```python
  my_list = [1, 2, 3]
  my_list.append(4)  # Adds an element
  ```

### 6. Tuples
- Immutable ordered collections.
- Example:
  ```python
  my_tuple = (1, 2, 3)
  print(my_tuple[1])  # Accessing elements
  ```

### 7. Sets
- Unordered collections of unique elements.
- Example:
  ```python
  my_set = {1, 2, 2, 3}
  print(my_set)  # Duplicates removed
  ```

### 8. Dictionaries
- Key-value mappings.
- Example:
  ```python
  my_dict = {"name": "Alice", "age": 25}
  print(my_dict["name"])  # Accessing values
  ```

### 9. Conditionals
- Control flow using `if`, `elif`, and `else`.
- Example:
  ```python
  age = 18
  if age >= 18:
      print("Adult")
  ```

### 10. Loops
- `for` and `while` loops.
- Example:
  ```python
  for i in range(3):
      print(i)  # Prints 0, 1, 2
  ```

### 11. Functions
- Define reusable blocks of code.
- Example:
  ```python
  def greet(name):
      return f"Hello, {name}!"
  print(greet("Alice"))
  ```

### 12. Modules
- Importing built-in and custom modules.
- Example:
  ```python
  import math
  print(math.sqrt(16))  # Square root
  ```

### 13. List Comprehension
- Concise way to create lists.
- Example:
  ```python
  squares = [x*x for x in range(5)]
  ```
